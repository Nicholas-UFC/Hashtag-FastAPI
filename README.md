# Hashtag FastAPI

Este projeto foi desenvolvido como estudo prático com base em aulas sobre **FastAPI**.

A ideia foi aplicar, na prática, conceitos como:
- criação de rotas
- organização do projeto em módulos
- autenticação com token
- modelagem de dados com SQLAlchemy
- uso de schemas com Pydantic

O código foi estruturado principalmente dentro da pasta `src`, separando:
- `routes` para as rotas da API
- `models` para modelos e schemas
- `utils` para funções auxiliares e dependências

Comando para iniciar o uvicorn: 
uvicorn src.main:app --reload
